Installing Apps

To install apps on Window 10 laptops, you use the Windows Stores. Listed below is the step by step instructions on how to download an app the Windows Store!
  1. Tap the Windows icon to open the Start menu.
  2. Click the Windows Store icon.
  3. Sign in using your Microsoft account.
  4. Find the app you wanted to download then double click on download.
However, if the software/program your looking for isn't on the Window store, there's another set of instructions to follow.
  1. Find the software you want to download and double-click on the download button/link.
  2. Then go to recently"downloads" and click to open.
  3. Then a box will pop up asking if you want to open it and select yes.
