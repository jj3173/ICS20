Summative Game

Requirements

These are a few requirements for my game:

  • The characters in my game are going to be Mario (controled by the player) and the Bullet (who is the opponent).
  • The score will be displayed on the right side of the screen and will be tracked by the time Mario can last before dying.
  • The starting screen of the game is title of the game and a start button underneath. Mario will also appear on the screen to help the player visualize how to control Mario.
  • When the player is ready to start the game, they can tap anywhere on screen with the mouse.
  • The point of this game is to dodge all the Bullets by controling Mario with the set of arrows and the keypad.
  • If the Bullet hits Mario, the Game Over screen will be displayed.
