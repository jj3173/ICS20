Sharing Information

To share documents on Lenovo laptops to other laptops that have Windows 10, you can use OneDrive, advanced sharing, or an USB. The instructions for each method is listed bellow!

Method 1

This method is using OneDrive.

Image Belong To OneDrive

  1. Install OneDrive if not already on your laptop (if you don't know how check out the previous page)
  2. Log into your Microsoft account.
  3. Open a tab for the OneDrive folder.
  4. Select documents/files you want to share.
  5. Drag /or/ copy paste them into the Ondrive folder.
  6. Now you can access these files on other computers if you log into your OneDrive!

Method 2

This method is used to Advanced Sharing.

  1. Right click on which ever file/document you need.
  2. Select the share with button and then choose advanced sharing.
  3. Then modify the options to your preference. You can vary many aspects ranging from number of simultaneous users accessing the shared folder to the share permissions.

For More Information Visit howtogeek.com

Method 3

Another way to share between any computers is to use an USB stick.

    Image Belong To usbstick.co.uk
  1. Insert the USB in the computer.
  2. Press computer logo and click the USB option.
  3. Open another window to select whatever files you want to move.
  4. Drag each file into the USB window and wait for it to copy.
  5. Right click and select remove USB stick.

Sharing Printer

This is how to share a printer that is connected to a PC without using a Homegroup.

  1. Click the window button and search printers and devices (located on the control panel).
  2. Right click the pinter you want to share and select the printer properties.
  3. On the printer properties tab select sharing where you can varie the properties to satisfy your needs.

Another way is to use Homegroup.

  1. Start off by finding the Homegroup (located on the control panel).
  2. Select the change what your sharing on Homegroup link and turn printers and devices on.
  3. Your printer is then shared with all the Homegroup members. However, if they aren't an member there are more steps. Here's the link where you can find more information, howtogeek.com
